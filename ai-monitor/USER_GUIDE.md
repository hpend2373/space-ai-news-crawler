# Brief 앱 사용 가이드 📱

## 🎯 첫 실행

### 1. 앱 시작
앱을 실행하면 **아름다운 모드 선택 화면**이 나타납니다.

```
┌─────────────────────────────────────┐
│        ✨ Brief                     │
│   어떤 뉴스를 확인하시겠습니까?      │
│                                     │
│  🚀 Stock + Space    🤖 AI News    │
│  우주 산업 & 주식     OpenAI, Anthropic│
│                                     │
└─────────────────────────────────────┘
```

### 2. 모드 선택
원하는 카드를 클릭하세요!

- **🚀 Stock + Space Brief**: SpaceX, Blue Origin, Rocket Lab 등
- **🤖 AI News Brief**: OpenAI, Anthropic, Google AI

### 3. 폴더 접근 허용
처음 실행 시 **폴더 접근 권한** 요청이 나타납니다.

```
⚠️ 폴더 접근 권한 필요
앱이 정상적으로 작동하려면 스크립트 폴더에 대한 접근 권한이 필요합니다

[허용] 버튼 클릭
```

## 📊 데이터 보기

### 카드 뷰 (기본)
뉴스 항목이 **카드 형태**로 표시됩니다.

```
┌─────────────────────────────────────┐

│                                     │
│ 📰 15건 기사 | 🚨 3건 이슈          │
│                                     │

│                                     │
│ • Starship achieves orbit           │
│ • FAA approval granted              │

└─────────────────────────────────────┘
```

#### 카드 상태

- **노란색 별**: 중요도 높음 (AI News)

### 카드 확장
카드를 **클릭**하면 상세 정보가 펼쳐집니다.

## 🔄 워크플로우 예시

### Stock + Space Brief 사용

1. **앱 시작** → Stock + Space 선택
2. **허용** → 폴더 접근 권한 부여
3. **📡 정보 수집** 클릭
   ```
   🔄 수집 실행 중...
   ⏳ Python 스크립트 실행
   ✅ 수집 완료!
   ```
4. **자동 새로고침** → 카드 표시
5. **카드 클릭** → 상세 정보 확인
6. **🏠 홈** 클릭 → AI News로 전환

### AI News Brief 사용

1. **모드 선택** → AI News 선택
2. **📡 AI 뉴스 수집** 클릭
3. **데이터 확인** → OpenAI, Anthropic, Google AI
4. **별 표시** 확인 → 중요도 높은 뉴스
5. **확장** → 원문 링크 클릭

## 🎨 UI 특징

### 호버 효과 🖱️
마우스를 카드 위에 올리면:
- 카드가 **살짝 확대** (1.01배)
- 그림자가 **진해짐**
- 테두리가 **강조됨**

### 애니메이션 ✨
- **부드러운 전환**: 화면 전환 시 슬라이드 + 페이드
- **스프링 효과**: 버튼 클릭 시 탄력적인 애니메이션
- **점진적 표시**: 카드 확장 시 부드럽게 펼쳐짐

### 상태 표시 🚦
- **초록색**: 수집 완료
- **파란색**: 수집 중
- **빨간색**: 오류 발생
- **회색**: 대기 중

## 🔧 고급 기능

### 번역 (Stock + Space)
Ollama qwen3:8b 모델로 **자동 번역**:
- 영어 뉴스 → 한국어 변환
- 백그라운드에서 자동 실행
- 원문도 함께 표시

### 자동 감지 (Stock + Space)
- **이슈 기준 충족**: 빨간색 강조
- **우주경제 섹션**: 보라색 테두리
- **중요도 점수**: 자동 계산

### 점수 시스템 (AI News)
- **4점 이상**: ⭐ 별 표시
- **프로바이더별 색상**:
  - OpenAI: 초록색 🟢
  - Anthropic: 오렌지 🟠
  - Google AI: 파란색 🔵

## 🐛 문제 해결

### "데이터를 불러오는 중..."이 계속 표시됨
→ 아직 데이터 파일이 없습니다. **정보 수집** 버튼을 클릭하세요.

### 수집 버튼을 눌러도 변화 없음
→ Xcode 콘솔(⌘Y)에서 오류 확인

### 번역이 안 됨
→ Ollama 서버 실행 확인:
```bash
ollama serve
```

### 홈 버튼이 안 보임
→ 앱을 다시 빌드하세요 (⌘B)
