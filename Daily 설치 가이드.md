# Daily 앱 설치 가이드

## 시스템 요구사항
- macOS 13 (Ventura) 이상
- Intel 또는 Apple Silicon Mac 모두 지원 (Universal Binary)

## 설치 방법

### 1단계: DMG 열기
`Daily.dmg` 파일을 더블클릭하여 마운트합니다.

### 2단계: 앱 복사
열린 창에서 `Daily.app`을 `Applications` 폴더로 드래그합니다.

### 3단계: 처음 실행 (중요!)
이 앱은 Apple Developer 공증 없이 배포되었기 때문에, 처음 실행 시 macOS Gatekeeper가 차단할 수 있습니다. 아래 방법으로 실행하세요.

**방법 A: 우클릭으로 열기**
1. Finder에서 Applications 폴더를 엽니다
2. `Daily` 앱을 **우클릭(또는 Control+클릭)**합니다
3. 메뉴에서 **"열기"**를 선택합니다
4. 경고 팝업에서 **"열기"** 버튼을 클릭합니다
5. 이후부터는 정상적으로 더블클릭으로 실행할 수 있습니다

**방법 B: 터미널 명령어 (방법 A가 안 될 경우)**
터미널을 열고 아래 명령어를 입력합니다:
```
xattr -cr /Applications/Daily.app
```
이후 앱을 더블클릭하면 정상 실행됩니다.

**방법 C: 시스템 설정에서 허용**
1. 앱을 더블클릭하면 차단 메시지가 나타납니다
2. **시스템 설정 → 개인 정보 보호 및 보안**으로 이동합니다
3. 하단에 "Daily 앱이 차단되었습니다" 메시지가 보입니다
4. **"확인 없이 열기"** 버튼을 클릭합니다

## 앱 정보
- 앱 이름: Daily
- 번들 ID: fastq.ai-app
- 버전: 1.0
- 아키텍처: Universal (arm64 + x86_64)

## 문제 해결
- **"손상되었으며 열 수 없습니다" 오류**: 터미널에서 `xattr -cr /Applications/Daily.app` 실행
- **앱이 바로 종료되는 경우**: macOS 버전이 13 이상인지 확인
